Instructions

This script validates a set of strings based on the rules below. The strings are separated by spaces.

  1. There are 15 valid characters: the lower-case letters 'a' through 'j' and the uppercase letters 'Z', 'N', 'L', 'Q', and 'R'.
  2. Every valid lower-case letter by itself is valid, for example, 'j' is a valid set of string, but 'fj' is not.
  3. If ♥ is a valid set of strings, then so is Z♥.
  4. If ♥ and ♦ are valid sets of strings, then so are N♥♦, L♥♦, Q♥♦, and R♥♦.
  5. All other strings are invalid.
